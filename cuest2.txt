1.- Describe qué es lo que realizan las pseudo-clases :hover y :link

Hover se da cuando pasa el mouse encima de la etiqueta que tenga esta pseudo-clase y link muestra enlaces no visitados

2. Menciona la diferencia entre un selector descendente y entre un selector hijo

Un selector descendente solo aplica para la etiqueta que se esta hablando dentro de otra etiqueta y el selector hijo hereda lo mismo que el padre

4.- Describe cómo funciona la herencia en CSS

5.-Explica brevemente el modelo de cajas de CSS

El modelo de cajas es el comportamiento de CSS que hace que todos los elementos de las páginas se representen mediante cajas rectangulares. Las cajas de una página se crean automáticamente.

6.-Describe de manera resumida en qué consisten (cómo funcionan) los selectores de clase, de id, de descendencia, de hijo

Los de clase sirven para que varias etiquetas sean seleccionados, los id sirven para una etiqueta en especifico, un selector descendente solo aplica para la etiqueta que se esta hablando dentro de otra etiqueta y el selector hijo hereda lo mismo que el padre
