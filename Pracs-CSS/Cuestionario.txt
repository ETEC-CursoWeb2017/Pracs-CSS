1.-Describe por lo menos tres etiquetas de web semántica
	+ <header> --> Va al inicio de la página web, aquí va un logotipo o una breve descripción, por ejemplo: ¿Quiénes somos?

2.- Definición de CSS

3.- Explica el modelo conceptual de funcionamiento de CSS
	+1) Analiza documento fuente y crea un árbol de documentos.
	 2) Identifica tipo de medio
	 3) Recupera hojas de estilo asociadas
	 4) Anota elementos de la estructura del documento
	 5) Asignación de valor único para cada propiedad.
	 6) Mecanismos de cascada y herencia
	 7) Generar una estructura de formato
	 8) Transfiere estructura de formato
	 9) El soporte del destino
