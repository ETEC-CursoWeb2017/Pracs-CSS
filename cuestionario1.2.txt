2.- Definición de CSS
	Cascading Style Sheets
3.- Explica el modelo conceptual de funcionamiento de CSS
	CSS analiza el maquetado, y crea un árbol de documentos, identifica cada tipo de medio y recupera hojas de estilo 	asociadas,identifica los elementos del maquetado, asigna un sólo valor a cada propiedad,en forma de cascada y 	herencia genera un tipo formato y lo transfiere al navegador.
4.- ¿Qué es una regla y cuáles son sus componentes?
	Una regla está conformada por un selector que tiene una propiedad y un valor
5.- Menciona la importancia de CSS y sus principios
